REST API for books management using Gin and gorm.
This REST API includes CRUD operations.

Server: localhost:8080
Handlers:
GET    /v1/
GET    /v1/hello
GET    /v1/books/:id/:title
GET    /v1/query
POST   /v1/books
GET    /v1/books
GET    /v1/book/:id
PUT    /v1/book/:id
DELETE /v1/book/:id
GET    /v2/buku/:kode/:judul
GET    /v2/query
